# How to Run and Use the PDF Chat App - Simple Guide 🚀📚

## 📋 What You Need First
- A computer with Windows, Mac, or Linux
- Python installed (version 3.8 or newer)
- Internet connection (only for first-time setup)

## 🎯 Step 1: Download the Code
1. Go to GitHub and download this project
2. Save it to your Desktop in a folder called `pdf_chat_project`
3. You should have these files:
   - `chat_with_multiple_pdfs.py` (main app)
   - `requirements.txt` (list of needed packages)

## 🖥️ Step 2: Open Terminal/Command Prompt
**Windows:** Press Windows key + R, type `cmd`, press Enter
**Mac:** Press Cmd + Space, type `terminal`, press Enter
**Linux:** Press Ctrl + Alt + T

## 📁 Step 3: Go to Your Project Folder
Type this in terminal:
```cmd
cd Desktop/pdf_chat_project
```

## 🔧 Step 4: Set Up Virtual Environment
Copy and paste these commands one by one:

```cmd
python -m venv pdf_env
pdf_env\Scripts\activate
```

**Success sign:** You should see `(pdf_env)` at the beginning of your terminal line

## 📦 Step 5: Install Required Packages
This will take 5-10 minutes. Copy and paste:

```cmd
pip install streamlit PyPDF2 langchain langchain-community langchain-huggingface transformers sentence-transformers torch faiss-cpu accelerate numpy pandas
```

Wait for it to finish downloading and installing everything.

## 🚀 Step 6: Start the App
Type this command:
```cmd
streamlit run chat_with_multiple_pdfs.py
```

**What happens:** 
- Your web browser will automatically open
- You'll see the app at `http://localhost:8501`
- If browser doesn't open, manually go to that address

## 📚 Step 7: How to Use the App

### Upload Your PDFs:
1. Look for the sidebar on the left
2. Click "Choose PDF files"
3. Select one or more PDF files from your computer
4. Click "🔄 Process Documents"

### Wait for Processing:
- First time: Takes 2-3 minutes (downloading AI models)
- Next times: Takes 30 seconds to 1 minute
- You'll see progress bars showing what's happening

### Start Chatting:
1. After processing is done, you'll see a chat box at the bottom
2. Type your question about the PDFs
3. Press Enter
4. Wait for the AI to respond (takes 5-15 seconds)
5. You'll get an answer with references to which part of your PDF was used

## 💬 Example Questions You Can Ask:
- "What is this document about?"
- "Summarize the main points"
- "What are the conclusions?"
- "Explain the methodology"
- "What recommendations are given?"

## 🎯 Daily Usage (After First Setup)
Every time you want to use the app:

1. Open terminal/command prompt
2. Type: `cd Desktop/pdf_chat_project`
3. Type: `pdf_env\Scripts\activate`
4. Type: `streamlit run chat_with_multiple_pdfs.py`
5. Browser opens automatically

## ✅ How to Know It's Working
- Terminal shows `(pdf_env)` before your command line
- Browser opens automatically to the app
- You can see the file upload area on the left side
- No error messages in the terminal

## 🛑 How to Stop the App
- Go back to your terminal window
- Press `Ctrl + C`
- Type `deactivate` to exit the virtual environment

## 🔧 If Something Goes Wrong

### Problem: "python not found"
**Solution:** Install Python from python.org

### Problem: "streamlit not found"  
**Solution:** Make sure you activated the virtual environment:
```cmd
pdf_env\Scripts\activate
```

### Problem: App won't start
**Solution:** Try these commands:
```cmd
pip install --upgrade streamlit
streamlit run chat_with_multiple_pdfs.py
```

### Problem: Can't process PDFs
**Solution:** Make sure your PDFs have text (not just images)

## 📖 What the App Does
1. **Reads your PDFs** - Extracts all text from your documents
2. **Breaks into chunks** - Splits text into smaller pieces for processing
3. **Creates vectors** - Converts text to numbers AI can understand
4. **Stores in database** - Saves everything for quick searching
5. **Answers questions** - Uses AI to find relevant information and respond
6. **Shows sources** - Tells you exactly where the answer came from

## 💡 Pro Tips
- **Start small:** Try with 1-2 PDFs first
- **Good PDFs:** Use documents with clear, readable text
- **Be specific:** Ask detailed questions for better answers
- **Check sources:** Always verify the AI's answers with the source references
- **Keep terminal open:** Don't close the terminal while using the app

You now have a personal AI assistant that can read and answer questions about any PDF documents you give it. It's completely free and works entirely on your computer!
